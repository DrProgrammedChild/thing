Toxic bot

Basic commands

-ping | Pings discord
-source | Some of the sexy programming behind the bot
-version | Version of the bot (and some legal stuff)
-website | Visit our lovely website

Fun commands

-play <song> | Plays a song
-pause | Pauses current song
-resume | Resumes current song
-disconnect | Disconnects from current voice channel
-queue | Displays queue
-skip | Skips current song in queue
-meme [deepfried] | Gives you a MEME

Currency commands

-tc | Displays how many toxic coins you (or someone else) has
-shop | DM's you a list of buyable items
-buy <itemnum> | Buys an item from the shop

Admin commands

-say <message> | Says something
-kick <user> | Kicks a user
-ban <user> | Bans a user
-addcoins <user> <amount> | Gives a user coins
-takecoins <user> <amount> | Takes coins from a user
-kill | Takes the bot offline
-eval <code> | Executes code
